Clear Cache
===========

Clears the complete TYPO3 Cache. Since TYPO3 6.x including OpCode Cache just like the Install Tool.

Call it like:

typo3/cli_dispatch.phpsh mklib -fc

or

typo3/cli_dispatch.phpsh mklib --flush-cache
